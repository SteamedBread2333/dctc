# dynamic-compile-tsx-commond(dctc)
Dynamically compile TSX/TS files and execute them.

## Usage
```
dctc [options] <file>
```

## Options
```
-h, --help      display help for command
-v, --version   output the version number
```

## Installation
```shell
npm install -g dynamic-compile-tsx-commond
```

## Example
```shell
dctc ./src/index.tsx
```
